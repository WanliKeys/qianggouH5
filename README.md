# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1pPi-TQjSvtt8NhtEcM4aOA21CIPnM1xU

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. 设置后端地址（可选）：在 `.env.local` 中增加 `VITE_API_BASE=http://localhost:3001`
3. Run the app:
   `npm run dev`
